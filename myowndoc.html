Principais Tags HTML

  • h1: Tag para cabeçalho nivel 1;

  • h2: Tag para cabeçalho nivel 2;

  • h3: Tag para cabeçalho nivel 3;

  • h4: Tag para cabeçalho nivel 4;

  • h5: Tag para cabeçalho nivel 5;

  • h6: Tag para cabeçalho nivel 6;

  • p: Tag para separar Paragrafos;

  • strong: Visualmente deixa o texto em negrito, quando um leitor de tela passa pela tag strong ele idenfifica a enfase gerada;

  • b: Visualmente deixa o texto em negrito, mas diferentemente da tag strong, nao gera a enfase no texto;

  • br: Pula uma linha;

  • ul: Tag para adicionar uma lista nao enumerada;

  • ol: Tag para adicionar uma lista enumerada;

  • li Adiciona um item a lista ul ou ol;

  • em: Visualmente deixa o texto em italico, quando um leitor de tela passa pela tag em ele idenfifica a enfase gerada;

  • i: Visualmente deixa o texto em italico, mas diferentemente da tag em, nao gera a enfase no texto;

  • a: linka uma url;

  • div: Define uma divisao ou secao no HTML;

  • table: Define uma tabela;

  • thead: Define o head da tabela;

  • tbody: Define o body da tabela;

  • tr: Define uma linha da tabela da tabela;

  • th: Define uma coluna do head da tabela (centrallizado e negrito);

  • td: Define uma coluna do body da tabela;

  • form: Define um formulario;

  • label:

  • input:
